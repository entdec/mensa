* [X] very fast

* [ ] filtering
* [ ] group by
* [ ] sum/max/min
* [ ] tables without headers (and without most of the above)
* [ ] tables backed by arrays (of ActiveModel)

## Usage
How to use my plugin.

## Installation
Add this line to your application's Gemfile:

```ruby
gem "mensa"
```

And then execute:
```bash
$ bundle
```

Or install it yourself as:
```bash
$ gem install mensa
```
